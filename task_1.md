"Certifications" (Сертификации)- это основной раздел, где представлена вся информация о программе сертификации тестировщиков (схема сертификации, описание каждого сертификата, учебныеи планы)
"SCR" (Реестр успешных кандидатов) - это раздел для проверки номера сертификата, предоставленного кандидатом
"Glossary" (Глоссарий) - это официальный словарь терминов по тестированию программного обеспечения.
"Отладка" (Debugging) - процесс поиска, анализа и устранения причин отказов компоненте или системе.
